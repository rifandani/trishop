## What is Trishop ‚ùì

Trishop is a free and open-source e-commerce platform for business units who want to find customers and sell their own products online. This platform is different from platforms such as Shopee, Tokopedia or the like which use a multi-role user system where users can register as buyers and sellers at the same time. This platform is meant for only one seller.

## What is inside ‚ùì

Trishop equipped with features:

1. Safe typed Next.js 12 (React 18) with Typescript + Eslint + Prettier best practices ‚ö°
2. E2E testing with Cypress + Cypress workflows ‚úÖ
3. UI components documentation with Storybook and deployed to Chromatic üå∏
4. Database using MongoDB and Mongoose ODM üçÉ
5. File storage using Cloudinary üíæ
6. Authentication using JWT inside of cookie üîë
7. Role-based authorisation: `[USER, ADMIN]` _for now_ üîê
8. Form validation with Formik and Yup ‚ú®
9. Customer cart and wishlist with React Context _(to be replaced by redux)_ üõ†

## How to use üèÄ

To run this project locally, follow these simple steps:

1. Clone repo
2. Run `yarn install` or `npm install`
3.
4. Run `yarn dev` or `npm run dev` to run Next.js frontend and backend in port `3000`

Notes:

1. Backend API is in `/pages/api/v1` folder
2. Run `yarn storybook` to run Storybook in port `6006`
